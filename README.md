# Dependencies
- x264  
```
sudo apt-get install libx264-dev
```
- faac  
```
sudo apt-get install libfaac-dev
```
- librtmp  
```
sudo apt-get install librtmp-dev
```
- alsa  
```
sudo apt-get install libasound2-dev
```
- v4l2  
- opencv3 (optional)

# Compile  
```
mkdir build
cd build
cmake ..
make
```
